# Reflection

One of the main challenges in this assignment was selecting the most appropriate GitHub project template. While multiple templates exist, understanding which one aligns best with Agile methodology required careful consideration.

Customizing the Kanban board also presented challenges, particularly in ensuring that it accurately reflects real-world workflows. Adding columns such as Testing and Blocked helped improve clarity but required thoughtful structuring.

Another challenge was linking previous assignment work into the board. Ensuring traceability from Assignment 6 user stories to the Kanban board required revisiting earlier work and aligning tasks properly.

Compared to tools like Trello and Jira, GitHub Projects is more integrated with development workflows. However, it is slightly less intuitive for beginners. Trello offers simplicity, while Jira provides advanced features. GitHub strikes a balance by integrating directly with code and issues.

Overall, this assignment improved my understanding of Agile project management and the importance of visual workflow tracking.
