# Template Analysis

| Template | Columns | Automation | Agile Suitability |
|----------|--------|------------|------------------|
| Basic Kanban | To Do, In Progress, Done | None | Simple workflows |
| Automated Kanban | To Do, In Progress, Done | Auto-moves issues | Strong Agile support |
| Bug Triage | Needs Review, Priority | Moderate | Debugging-focused |

## Selected Template: Automated Kanban

### Justification

The Automated Kanban template was selected because it supports Agile development through workflow automation. Issues automatically move across stages, reducing manual effort.

It aligns with sprint-based development from Assignment 6 and allows continuous tracking of user stories and tasks.

Additionally, it improves visibility and efficiency when managing multiple tasks.
