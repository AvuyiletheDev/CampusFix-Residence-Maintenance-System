# Kanban Board Explanation

A Kanban board is a visual project management tool used to track work across different stages of a process.

## Workflow Visualization

The board uses columns such as To Do, In Progress, Testing, Blocked, and Done to represent task progression.

## Work-in-Progress (WIP)

To avoid overload, tasks are limited in each stage to ensure focus and efficiency.

## Agile Support

Kanban supports Agile principles by allowing continuous delivery and adaptability. Tasks can be updated dynamically without strict sprint constraints.

## Customization

Additional columns such as Testing and Blocked were added to reflect real-world development challenges and improve workflow tracking.
